# Rule: Premium UI/UX 🎨

## Goal

Ensure the application feels premium, modern, and alive.

## Guidelines

- **Rich Aesthetics**: Use vibrant but harmonious color palettes. Avoid "flat" designs if depth (shadows, gradients, glassmorphism) adds value.
- **Micro-animations**: Implement subtle transitions and hover effects to make the interface feel responsive.
- **Responsiveness**: Innovation must work on all screen sizes. Prioritize mobile-first but don't neglect the desktop experience.
- **Accessibility**: Beautiful design must also be inclusive. Use Radix UI primitives to ensure high accessibility standards without sacrificing style.
- **Creative Freedom**: You are encouraged to iterate on UI components to make them "WOW" the user, as long as they follow the project's core identity.
